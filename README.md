# Super Store Sales Dashboard

## Project Overview
This project is an interactive Power BI dashboard created to analyze Super Store sales performance, customer purchasing behavior, regional trends, and profit insights. The dashboard helps in understanding business growth and decision-making through data visualization techniques.

## Tools & Technologies Used
- Power BI
- Microsoft Excel / CSV Dataset
- Data Visualization
- Data Cleaning & Transformation

## Dashboard Features
- Sales, Profit, Quantity, and Delivery KPI Cards
- Region-wise Sales Analysis
- Sales by Category and Sub-Category
- Sales by Payment Mode
- Sales by Segment
- Monthly Sales and Profit Analysis (Year-over-Year)
- State-wise Sales Visualization using Maps
- Ship Mode Analysis
- Interactive Region Filter/Slicer

## Key Insights
- Highest sales were generated from Office Supplies category.
- Chairs and Phones were among the top-performing sub-categories.
- Standard Class shipping mode had the highest sales contribution.
- Sales showed significant growth during the final quarter of the year.
- California and New York generated the highest sales among states.
